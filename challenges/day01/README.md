# Day 01: JavaScript基础 - 变量、作用域与闭包

## 🎯 今日目标

掌握JavaScript中最重要的基础概念：变量声明、作用域和闭包。这些概念是理解JavaScript工作原理的基础，也是面试中的高频考点。

## 📚 学习内容

### 1. 变量声明的三种方式
- `var`、`let`、`const`的区别
- 作用域规则
- 变量提升（Hoisting）
- 暂时性死区（Temporal Dead Zone）

### 2. 作用域理解
- 全局作用域
- 函数作用域  
- 块级作用域
- 词法作用域

### 3. 闭包概念
- 什么是闭包
- 闭包的实际应用
- 常见的闭包陷阱

## 💡 实践挑战

### 挑战1：变量声明练习
创建一个函数，演示`var`、`let`、`const`在不同作用域下的行为差异。

### 挑战2：闭包计数器
实现一个使用闭包的计数器，具有以下功能：
- 增加计数
- 减少计数
- 获取当前计数
- 重置计数

### 挑战3：模块模式
使用闭包实现一个简单的模块模式，创建一个用户管理模块。
